# 📈 StockPulse - Full Stack Zerodha Clone

**StockPulse** is a professional-grade trading platform inspired by Zerodha (Kite). It features a modern marketing landing page, a functional user trading dashboard, and a robust Node.js backend to manage stock portfolios and real-time data visualization.

---

## 🚀 Live Demo
* 🌐 **Landing Page:** [stockpulse0.netlify.app](https://stockpulse0.netlify.app/)
* 📊 **Trading Dashboard:** [dashborad03.netlify.app](https://dashborad03.netlify.app/)

---

## ✨ Features

### **1. Marketing Landing Page (Frontend)**
* Built with **Vite** for high performance and SEO optimization.
* Fully responsive design for Home, About, Pricing, and Support pages.
* Professional UI components inspired by the official Zerodha website.
### **2. Trading Dashboard**
* Interactive **Portfolio Tracking** with a clean, focused UI.
* **Dynamic Charts:** Integrated with **Chart.js** to visualize holdings and investment trends.
* **Watchlist & Order Execution:** Features functional Buy/Sell windows to simulate the trading experience.
* **Real-time Calculations:** Automatic profit/loss and margin calculations.

### **3. Backend & API**
* RESTful API built with **Node.js & Express**.
* **MongoDB Atlas** integration for secure data persistence.
* Schema-based management for Orders, Holdings, and Positions.

---
## 🛠️ Tech Stack

- **Frontend:** React.js, Vite, Bootstrap, CSS3.
- **Dashboard:** React.js (CRA), Material UI, Chart.js.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB Atlas (Mongoose).
- **Deployment:** Netlify (Client-side), Render (Server-side).
